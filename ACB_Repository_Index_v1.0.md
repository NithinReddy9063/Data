# APEX Commercial Bank Repository Index

## 1 Scope Control

This repository index reserves the complete Version 1.0 document population for the APEX Commercial Bank knowledge base. It does not generate the controlled document bodies. Each reserved record must be authored later using the approved master template for its document type.

The phrase "All approved products in ACB Knowledge Base Bible v1.0" resolves only to the frozen product list in ACB-KB-BIBLE-001. The phrase "All approved customer types in ACB Knowledge Base Bible v1.0" resolves only to the frozen customer type list in ACB-KB-BIBLE-001.

For multi-word domains, the document ID uses the approved domain label directly because no separate domain-code register has been approved.
